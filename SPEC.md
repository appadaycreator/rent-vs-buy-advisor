# 賃貸vs購入どちらがお得診断【無料】無料診断・アドバイス - 技術仕様書

## 概要

**サービス名**: Rent Vs Buy Advisor
**バージョン**: 2.0.0
**更新日**: 2026-05-29
**URL**: https://appadaycreator.com/rent-vs-buy-advisor/

家賃・物件価格・居住年数・ライフスタイルから賃貸と購入のトータルコストを比較診断。登録不要・完全無料でご利用いただけます。

## v2.0.0 主な変更点（2026-05-29）

- **入力項目拡充**: 頭金・管理費+修繕積立金（月額）を追加。購入時諸費用（物件価格×6%）を自動計算
- **損益分岐点計算**: 「何年住めば購入が有利になるか」を1〜50年ループで自動算出・表示
- **ビジュアル比較チャート**: Canvas APIで賃貸/購入のトータルコストを横棒グラフで可視化
- **結果UIの刷新**: テキストベースからカード式2列レイアウトに変更。月額換算・内訳を明確化
- **スライダーUI**: 居住年数・金利をスライダー+数値入力で直感的に調整可能
- **localStorage自動保存・復元**: 前回入力値をブラウザに保存、次回アクセス時に自動復元
- **計算精度改善**: 資産残存価値を60%→70%に修正。修繕費単価を見直し。ローン元本から頭金を控除

## 計算式

| 項目 | 計算方法 |
|------|---------|
| 賃貸トータル | 月家賃×12×年数 + 30万（引越等） |
| ローン元本 | 物件価格 - 頭金 |
| ローン総返済 | 元利均等計算（月次複利） |
| 購入時諸費用 | 物件価格×6%（仲介料・登記・税等） |
| 固定資産税 | 物件価格×1.4%×年数 |
| 自己修繕費 | 年8万円×年数 |
| 管理費合計 | 管理費+修繕積立金（月額）×12×年数 |
| 資産残存価値 | 物件価格×70%（売却時残存） |

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **保存キー**: `rvb_inputs_v2`（rent/price/down/mgmt/years/rate）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- Canvas API（棒グラフ描画）
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. 月家賃・物件価格・頭金・管理費を入力する
2. 居住予定年数・ローン金利をスライダーで調整する
3. 「賃貸vs購入を比較する」ボタンをクリックする
4. コスト内訳・グラフ・損益分岐点を確認する
5. 条件を変えて複数パターンを比較する（前回値が自動復元されます）
